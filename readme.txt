Di sini, saya belajar cara menerapkan 3D object dalam website lewat Three.js

Pada kamera default, user dapat menggerakan piring (plate) dan jeruk (orange).
User tinggal klik salah satu benda tersebut sekali.
Sekarang, objek tersebut akan mengikuti mouse cursor.
Klik lagi untuk menaruhkan objek pada meja tersebut.

User dapat pencet spasi untuk pindah ke kamera lainnya, yang ada orbit control.
User sekarang dapat pencet "s" untuk memutarkan burger.
User dapat pencet spasi lagi untuk pindah ke kamera default.

Cylinder yang bergerak di lantai dimaksudkan untuk menjadi semacam robot vacuum.
